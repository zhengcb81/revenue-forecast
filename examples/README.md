## 典型使用场景

### 场景1: 产品驱动型公司
**适用公司**: 消费品、汽车、家电、电子产品
**关键分析点**:
- 品牌矩阵分析（Step4.5）
- 产品线扩张
- 市场份额变化
- 定价能力

### 场景2: 基础设施驱动型公司
**适用公司**: 电力、水务、燃气、高速公路
**关键分析点**:
- RAB监管资产基数
- 准许收益率（8-10%）
- 用户渗透率
- 价格调整机制

### 场景3: 项目制驱动型公司
**适用公司**: 建筑、造船、EPC、军工
**关键分析点**:
- 在手订单/营收比
- 订单获取能力
- 项目交付周期
- 现金回款情况

### 场景4: 房地产/REITs驱动型公司
**适用公司**: 商业地产、物流地产、REITs
**关键分析点**:
- Cap Rate（资本化率）
- NOI（净营业收入）
- 出租率
- WALE（加权平均租约期限）

## 报告结构示例

一份完整的分析报告包含以下章节：

1. **执行摘要** - 核心结论、投资评级、关键驱动因素
2. **关键财务指标预测** - 营收、利润、现金流预测
3. **双曲线业务分析** - 第一/第二/第三曲线分析
4. **蒙特卡洛模拟** - 概率分布、VaR、敏感性分析
5. **压力测试** - 极端情景评估、生存概率
6. **ESG风险调整** - E/S/G三维评估
7. **情景分析** - 乐观/基准/悲观三种情景
8. **跨公司比较** - 行业对比、排名、投资建议矩阵
9. **投资建议** - 评级、目标价、风险提示
10. **参数溯源** - 数据来源、假设说明
11. **准确度追踪** - 历史预测回顾、校准建议
