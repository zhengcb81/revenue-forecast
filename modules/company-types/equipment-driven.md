##### 4.6 设备驱动型公司增长驱动因素分解 ⭐️⭐️⭐️ 新增（2025-01-10）

**核心概念：** 半导体设备、工业母机、医疗设备等设备驱动型公司的增长逻辑与消费品公司有本质区别。传统"扩品类×扩地域×价格提升"框架**不适用**于设备公司，需采用专属分析框架。

**特别说明**：对于具备国产替代爆发式增长潜力的设备公司（如珂玛科技的静电卡盘、陶瓷加热器），在通过客户验证后可能呈现非线性爆发式增长。这类公司除适用本框架外，还需参考**4.7节爆发式增长分析框架**进行专项分析。

---

###### 4.6.1 设备公司与消费品公司增长逻辑对比

| 维度 | 消费品公司 | 设备驱动型公司 |
|-----|-----------|---------------|
| **增长来源** | "更多人买+买更多品类+卖更贵" | "同一客户买更先进设备+突破更高端客户" |
| **品类边界** | 清晰可分割（手机≠汽车） | 技术连续体（刻蚀→MOCVD→薄膜是同一技术栈） |
| **地域逻辑** | 地理市场（中国→印度→欧洲） | 客户层级（国内晶圆厂→台积电→英特尔） |
| **决策主体** | 消费者（C端） | 晶圆厂工程师/采购（B端） |
| **渠道模式** | 直销/经销/电商 | 纯直销，直供客户 |
| **订单周期** | 短（天-周） | 长（月-年） |
| **关键指标** | 市占率、ASP、复购率 | 技术领先度、客户认证、订单能见度 |

---

###### 4.6.2 设备驱动型公司增长公式

**核心公式：**
```
营收增长率 ≈ 技术突破贡献 + 客户升级贡献 + 产能释放贡献 + 国产替代贡献
```

**爆发式增长特例**：对于国产替代关键产品（如珂玛科技静电卡盘），一旦验证通过且产能充足，可能呈现非线性爆发式增长。此时需采用S型增长曲线建模：
```
营收增长率 ≈ S型渗透率曲线 × 产能约束 × 竞争反应因子
```
具体建模方法参见4.7节。

**四维驱动因素定义：**

| 驱动因素 | 含义 | 设备公司示例 |
|---------|------|-------------|
| **技术突破贡献** | 先进制程/新产品研发带来的增量 | 中微3nm刻蚀机突破 |
| **客户升级贡献** | 从二线客户→一线客户→国际客户的突破 | 中微进入台积电供应链 |
| **产能释放贡献** | 新工厂/基地投产带来的交付能力提升 | 中微临港基地投产 |
| **国产替代贡献** | 抢占外国设备厂商的市场份额 | 国产设备替代Lam/AMAT |
