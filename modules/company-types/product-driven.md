##### 4.5 销售额驱动因素分解 ⭐️⭐️⭐️ 重点补充（产品驱动型公司专项）

**核心概念：** 产品驱动型公司的营收增长可分解为三大驱动因素，通过量化各因素贡献度，可更准确地预测未来增长。

**销售额增长分解公式：**
```
营收增长率 = (1 + 扩品类贡献) × (1 + 扩地域贡献) × (1 + 价格贡献) - 1
         ≈ 扩品类贡献 + 扩地域贡献 + 价格贡献 + 交叉效应
```

---

```
综合CAGR = 分区域CAGR + 品牌贡献因子 + 技术追赶因子

示例（宝马集团）：
分区域CAGR = +0.85%
技术追赶因子 = -1.54%
品牌贡献因子 = +2%（基于品牌矩阵分析）

综合CAGR = 0.85% - 1.54% + 2% = +1.31%
```

**数据获取渠道：**

**关键假设验证：**
- 品牌定位是否清晰且互补？
- 协同效应是否可持续？
- 新品牌推出节奏是否合理？
- 品牌溢价能力是否稳定？
