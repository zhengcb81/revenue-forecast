######  3.9 渠道健康度与反身性效应 ⭐️⭐️ 重点补充

**核心概念：** 当公司采用经销商/代理模式时，渠道库存和社会库存会形成**反身性效应**——价格上涨期库存囤积放大需求，价格下跌期库存抛售抑制真实需求。此效应需在评分中充分体现。

#######  3.9.1 渠道结构分析
- 直销收入占比 vs 经销/代理收入占比
- 经销商数量、层级结构（一级/二级/多级代理）
- 前十大经销商销售额占比（客户集中度）
- 渠道模式易受反身性影响程度评估

| 渠道模式 | 反身性敏感度 | 扣分参考 |
|---------|-------------|---------|
| 纯直销模式 | 极低 | 0 |
| 直销为主（>70%） | 低 | -0.1 |
| 经销为主（>70%） | 高 | -0.3 |
| 经销+产品金融属性 | 极高 | -0.4 |

#######  3.9.3 反身性风险指数（HRI）计算

**HRI公式：**
```
HRI = 0.30×合同负债因子 + 0.25×价格因子 + 0.25×库存因子 + 0.20×渠道信心因子
```

**各因子计算方法：**

| 因子 | 权重 | 计算公式 | 评分范围 |
|-----|-----|---------|---------|
| 合同负债因子 | 30% | (合同负债增速/10) × 权重 | -5 ~ +3 |
| 价格因子 | 25% | (批价/出厂价 - 1) × 5 | 0 ~ +5 |
| 库存因子 | 25% | (实际周转天数/基准天数 - 1) × 5 | -3 ~ +3 |
| 渠道信心因子 | 20% | 主观评分（调研数据） | -2 ~ +2 |

**HRI解读与评分影响：**

| HRI区间 | 风险等级 | 反身性效应 | 确定性扣分 |
|---------|---------|-----------|-----------|
| HRI > 2 | 🟢 低风险 | 可忽略 | 0 |
| 0 < HRI ≤ 2 | 🟡 中风险 | 轻度影响 | -0.1 |
| -2 < HRI ≤ 0 | 🟠 高风险 | 中度影响 | -0.3 |
| HRI ≤ -2 | 🔴 极高风险 | 重度影响 | -0.5 |

**HRI计算示例（贵州茅台）：**
```
合同负债因子 = (-32%/10) × 10 = -3.2
价格因子 = (1.95 - 1) × 5 = +4.75
库存因子 = (100/60 - 1) × 5 = +3.33
渠道信心因子 = -0.5（调研偏弱）

HRI = 0.30×(-3.2) + 0.25×(+4.75) + 0.25×(+3.33) + 0.20×(-0.5)
    = -0.96 + 1.19 + 0.83 - 0.10
    = -7.68
```

**结论**：HRI = -7.68（🔴 极高风险），确定性扣分 -0.5

#######  3.9.4 反身性效应量化

- **正向放大效应（景气上行）**：价格↑ → 投机性囤货↑ → 表观需求↑ → 价格↑↑
  - 放大系数：1.2-1.5倍（表观需求/真实需求）

- **负向抑制效应（景气下行）**：价格↓ → 去库存抛售↑ → 真实需求被抑制 → 价格↓↓
  - 抑制系数：社会库存每增加1个月销量，抑制真实需求约8-12%

- **当前库存水位**：相当于X个月销量（需估算）
