## 三、 执行步骤

### 第一步：问题拆解与研究框架构建
当收到公司营收预测请求时，立即将任务拆解为以下9个研究维度：
1. 双曲线业务分析 ⭐️ 核心维度
1.1 第一曲线（传统业务/现金牛业务）

识别核心传统业务及其营收占比
传统业务的生命周期阶段（成长期/成熟期/衰退期）
传统业务的市场饱和度和增长天花板
传统业务面临的颠覆性威胁（技术替代、需求转移）
传统业务的现金流贡献和利润率趋势
传统业务能否通过效率提升、降本增效维持增长
预期传统业务的营收CAGR（未来5年）

1.2 第二曲线（新增长业务/战略业务）

识别公司正在培育的第二曲线业务
第二曲线业务的营收占比及增长速度
第二曲线业务的市场空间和渗透率
第二曲线业务的竞争格局和公司竞争优势
第二曲线业务从投入期到盈利期的时间表
管理层对第二曲线的资源投入力度（研发、资本开支）
第二曲线何时能成为主要营收驱动力
预期第二曲线业务的营收CAGR（未来5年）

1.3 曲线转换分析

第一曲线和第二曲线的营收占比变化趋势
两条曲线的接力时机（第二曲线能否在第一曲线衰退前成长起来）
是否存在第三曲线的早期布局（前瞻性）
业务转型的执行风险和不确定性

2. 宏观环境分析

全球及目标市场的GDP增长预测
所在国家/地区的货币政策和通胀趋势
相关行业的政策支持或监管变化
地缘政治风险对供应链的影响
周期性因素：所处行业是否有明显的经济周期性（如半导体、工程机械）

2.5 需求端驱动分析 ⭐️ 核心补充

#####  2.5.1 下游需求结构分析

- 各终端应用场景的营收占比及历史变化
- 主要需求来源（B端/G端/C端）及其增长趋势
- 核心客户群体的需求特征与购买行为
- 终端应用领域的生命周期阶段判断

#####  2.5.2 宏观经济关联性

- 公司营收与GDP、基建投资、消费指数等宏观经济指标的相关系数分析
- 历史数据回归分析（如茅台营收增速与基建增速的相关性）
- 宏观经济情景分析（乐观/基准/悲观）对下游需求的影响测算
- 货币政策、财政政策对需求的传导机制

#####  2.5.3 技术变革驱动的需求

- AI/新能源/物联网等新技术带来的**新增需求**规模与增速
- 技术迭代对现有需求的**替代效应**（如Mini-LED vs传统LED）
- 制程进步/产品升级对设备/材料的需求拉动强度
- 技术成熟度曲线位置对短期 vs 长期需求的影响

#####  2.5.4 政策与国家战略驱动的需求 ⭐️⭐️⭐️

**核心概念：** 某些行业需求主要来自政府政策、国家战略或大国竞争，而非纯粹的市场经济规律。这类需求具有**刚性、长期性和确定性**特征，是营收增长的重要支撑。

**分析框架：**

1. **大国竞争与国家安全需求**
   - 半导体：美国技术封锁倒逼国产替代，需求刚性极强
   - AI芯片：算力自主可控成为国家战略，需求确定性高
   - 军工/航空航天：国家安全需求，与军备预算直接挂钩
   - **需求特征**：高技术壁垒、时间紧迫、替代弹性小、订单周期长

2. **国家长期规划与产业政策**
   - 新能源汽车：国家2030碳达峰、2060碳中和目标驱动
   - 新能源（光伏/风电）：能源转型规划，年均装机量目标明确
   - 商业航天：纳入新基建，地方政府大力支持
   - **需求特征**：时限明确（5年/10年规划）、补贴支持、产能指标刚性

3. **国产替代与供应链安全**
   - 半导体设备/材料：2025年国产化率50%目标
   - 高端数控机床：自主可控要求，国产替代加速
   - 操作系统/工业软件：信创产业推动
   - **需求特征**：国产供应商优先、试错容忍度高、价格敏感度低

4. **政策强度评估**
   - **政策密集度**：中央+地方+部委联合发文数量
   - **财政支持力度**：补贴规模、税收优惠、专项贷款额度
   - **行政强制力**：是否纳入政府考核、行政指令强度
   - **时间约束**：是否有明确时间表（如2025年实现XX目标）

5. **政策可持续性判断**
   - **短期政策刺激**（如汽车购置税减免）：可能有周期性
   - **中期产业扶持**（如新能源补贴退坡）：关注补贴退出影响
   - **长期国家战略**（如半导体自主）：确定性最高，抗周期能力强

**案例应用（中微公司/半导体设备）：**
- **大国竞争驱动**：美国实体清单+技术封锁，国产替代需求刚性极强
- **国家战略支持**：国家集成电路大基金三期3440亿元投资，直接拉动设备采购
- **国产替代目标**：2025年国产化率50%目标，带来确定性增量需求
- **政策强度**：★★★★★（总书记亲自指示，部委联合推进）
- **可持续性**：★★★★★（长期战略，不受经济周期影响）
- **评分影响**：确定性调整额外+0.3分，需求端风险评估为极低

**案例应用（贵州茅台）：**
- 政策驱动需求极弱：市场化消费为主，受三公消费限制政策影响为负面
- 但需求稳定性高：非强政策驱动，反而需求更可持续（居民自饮消费）

#####  2.5.5 长期需求趋势（人口/消费/全球化）

- 人口结构变化（老龄化、Z世代消费力）对长期需求的影响
- 全球化/逆全球化趋势对国内外需求格局的影响
- 消费者偏好变迁（健康化、个性化、智能化）的影响

#####  2.5.6 季节性/周期性需求特征

- 需求的季节性规律及原因分析（如白酒春节旺季）
- 需求周期与行业产能周期的同步性/滞后性
- 库存周期（补库存/去库存）对短期需求的影响
- 下游客户资本开支周期对设备需求的领先性

#####  2.5.7 渠道库存与反身性效应（经销商/代理模式专项）⭐️⭐️

**核心概念：** 当公司采用经销商/代理模式时，渠道库存和社会库存会形成**反身性效应**——价格上涨期库存囤积放大需求，价格下跌期库存抛售抑制真实需求。

**分析框架：**

1. **渠道结构分析**
   - 直销收入占比 vs 经销/代理收入占比
   - 经销商数量、层级结构（一级/二级/多级代理）
   - 前十大经销商销售额占比（客户集中度）

2. **渠道库存周期**
   - 经销商平均库存周转天数（标杆：茅台约30-45天）
   - 历史渠道库存周期变化趋势
   - 公司当前渠道压货政策与强度
   - 渠道窜货现象监测

3. **社会库存评估（适用于茅台等具有金融属性产品）**
   - 终端消费者囤积行为分析
   - 二手市场交易活跃度与价格溢价
   - 产品金融属性强度（保值/增值预期）
   - 社会库存量估算（渠道库存+消费者库存）

4. **反身性效应量化**
   - **正向效应期（景气上行）**：价格↑ → 投机性囤货↑ → 表观需求↑ → 价格↑↑
   - **负向效应期（景气下行）**：价格↓ → 去库存抛售↑ → 真实需求被抑制 → 价格↓↓
   - **需求弹性计算**：社会库存每增加1个月销量，对真实需求的抑制系数

5. **库存风险指标**
   - 预收账款（合同负债）变化趋势：大幅下滑预示渠道信心不足
   - 应收账款周转天数增加（给经销商账期放宽）
   - 渠道价格倒挂（批发价跌破出厂价）
   - 电商平台/二手市场抛售量激增

**案例应用（贵州茅台）：**
- 直销占比提升（2024年近44%）**降低**渠道风险
- 2024年社会库存周转约3-4个月，处于**高位**
- 价格倒挂现象出现，表明经销商去库存压力
- 反身性效应测算：社会库存每减少1个月，短期销量影响约8-12%

**案例应用（半导体设备）：**
- 下游晶圆厂库存周期直接影响设备采购节奏
- AI驱动需求暴增期，客户存在超量下单行为
- 警惕"牛鞭效应"：下游需求波动在设备端被放大
- 订单能见度（Backlog）是判断真实需求的关键

3. 产业变革分析

AI技术对该行业的颠覆性影响
数字化转型的行业趋势
新技术(如自动化、IoT、新能源等)的应用前景
行业整体的技术成熟度曲线位置
行业是否处于范式转移期（Paradigm Shift）

4. 业务板块分析

各业务板块的历史营收占比和增长率
各板块所在细分市场的CAGR预测
各板块的利润率和现金流状况
新兴业务的增长潜力评估
业务组合稳定性：各板块增长是否同步，还是有显著分化

##### 2.X 渠道效率因子分析 ⭐️⭐️⭐️ v2.3.0新增

**核心概念**: 渠道网络是产品触达消费者的关键通道，渠道效率直接影响营收增长能力。

**渠道效率评估维度**：

| 评估维度 | 高分标准 | 低分标准 | 评分 |
|---------|---------|---------|------|
| **单店产出** | 高于行业均值50% | 低于行业均值30% | 1-5分 |
| **库存周转** | <30天 | >60天 | 1-5分 |
| **渠道覆盖密度** | 覆盖所有目标城市 | 仅覆盖一二线 | 1-5分 |
| **数字化水平** | 线上线下融合(OMO) | 纯线下传统渠道 | 1-5分 |
| **经销商盈利能力** | >行业均值 | 亏损或微利 | 1-5分 |

**渠道效率因子计算**：

```
渠道效率 = (单店产出 / 行业均值) × (基准周转天数 / 实际周转天数)

| 效率状态 | 效率因子 | CAGR调整 |
|---------|---------|---------|
| 高效率 | >1.2 | +1% |
| 基准 | 1.0 | 0% |
| 低效率 | <0.8 | -1% |

示例（宝马中国）：
- 单店产出 = 800辆/年（行业均值1000辆）→ 0.8
- 库存周转 = 45天（行业均值30天）→ 30/45 = 0.67
- 渠道效率 = 0.8 × 0.67 = 0.536 → 低效率
- CAGR调整 = -1%

应用公式：
调整后CAGR = 基础CAGR + 渠道效率因子
```

**渠道效率与营收增长的传导机制**：

```
高效率渠道 → 客户体验提升 → 转化率提高 → 复购率上升 → 营收增长
低效率渠道 → 库存积压 → 经销商亏损 → 积极性下降 → 服务质量下降 → 客户流失
```

**渠道问题预警信号**：

- 单店产出连续下降（>20%）
- 经销商库存周转>60天
- 经销商盈利门店占比<50%
- 渠道网络萎缩（门店数量减少）
- 线上渠道占比过低（<10%）但同行在20%+
