1. **必须严格按照CAGR查表,不得主观调整**

3. **不允许因为行业景气度高而提高评分**
4. **所有风险、收益因素都已在情景中反映,不得事后调整**

**验证检查点**:
```
步骤1: 计算综合CAGR
步骤2: 根据CAGR查表得到评分区间
步骤3: 线性插值计算精确评分
步骤4: ⚠️ 验证评分与CAGR匹配
步骤5: ⚠️ 如果评分在错误的区间,必须重新计算
步骤6: 输出评分时必须说明CAGR和对应的评分区间
```

**输出格式要求**:
```
评分: X.X分
CAGR: XX.X%
评分区间: XX% - XX% → X.X - X.X分

```

**设计要点**:
1. **拉开差距**: 每5个百分点CAGR约对应1分差距(更敏感)
2. **惩罚低速**: 15%以下CAGR评分快速下降(4.5-6.4分)
3. **奖励高成长**: 20%以上CAGR高分(7.5-9.4分)
4. **分段平滑**: 各区间内部线性插值,避免跳变
5. **⚠️ 严禁主观调整**: 评分必须100%由CAGR决定

**插值计算示例**:
- 综合CAGR = 19.2% → 评分 = 7.0 + (19.2-18)/(20-18) × 0.4 = **7.44分** ≈ 7.4分
- 综合CAGR = 15.6% → 评分 = 6.5 + (15.6-15)/(18-15) × 0.4 = **6.58分** ≈ 6.6分
- 综合CAGR = 20.4% → 评分 = 7.5 + (20.4-20)/(25-20) × 0.9 = **7.57分** ≈ 7.6分

**⚠️ 安踏体育评分示例(正确做法)**:
- 综合CAGR = 10.9%
- 评分区间: 10% - 12% → 4.5 - 5.4分
- 线性插值: 评分 = 4.5 + (10.9-10)/(12-10) × 0.9 = 4.5 + 0.45 = **4.95分** ≈ 5.0分
- 正确评级: ⚠️ **观望**
- ❌ **错误示范**: 给7.5分(这是20-25%CAGR对应的评分,严重违规!)

**投资评级说明**:

| 投资评级 | 评分范围 | 配置建议 | 持有周期 | 风险偏好 |
|---------|---------|---------|---------|---------|
| ✅ **强烈推荐** | ≥9.5分 | 30-40%(核心重仓) | 3-5年 | 能承受高波动 |
| ✅ **推荐** | 7.5-9.4分 | 20-30%(核心配置) | 3-5年 | 平衡型/进取型 |

#######  8.1.2 情景假设在评分中的体现 ⭐️⭐️⭐️

**核心概念**: 有利因素和不利因素已经通过情景反映在综合CAGR中,不再需要事后扣分。

**旧框架的问题**(v1.8.0):
```
基准情景CAGR = 19.4%(已考虑地缘政治风险)
→ 基准分 = 6.5分
→ 波动性扣分 = -0.65(地缘政治风险)
→ 最终得分 = 6.5 + (...) = 6.8分

问题: 地缘政治风险被双重惩罚!
```

**新框架的改进**(v2.0.0):
```
乐观情景(25%): CAGR = 26%(所有有利因素)
基准情景(50%): CAGR = 20.5%(正常情况)
悲观情景(25%): CAGR = 6.8%(所有不利因素,含地缘政治)
→ 综合CAGR = 20.4%
→ 直接查表: 评分 = 7.7分

优势: 风险只惩罚一次,在悲观情景中!
```

**情景中需要考虑的因素**:

**乐观情景有利因素**:
- ✅ 政策超预期(医疗新基建、国产替代加速)
- ✅ 业务突破(海外市场、新兴业务)
- ✅ 竞争格局改善(集采降价压力缓解)
- ✅ 技术突破(新产品快速放量)
- ✅ 地缘政治风险缓和

**基准情景正常情况**:
- → 按历史趋势和行业规律
- → 政策正常推进
- → 业务按计划发展
- → 宏观环境符合预期

**悲观情景不利因素**:
- ❌ 地产基建下行(影响医院采购预算)
- ❌ 地缘政治风险(美国制裁、核心零部件断供)
- ❌ 集采大幅降价(毛利率下降)
- ❌ 行业竞争加剧(价格战)
- ❌ 宏观经济下行
- ❌ 新业务拓展不及预期

**每个因素如何量化为CAGR影响?**

**示例1: 地缘政治风险对联影医疗的影响**
```
基准情景: 海外业务CAGR = 33%
悲观情景: 考虑美国制裁 → 海外业务CAGR = 3%(下降30个百分点)
影响: 拖累综合CAGR约3-5个百分点
```

**示例2: 地产基建下行对迈瑞医疗的影响**
```
基准情景: 国内生命信息与支持业务CAGR = 10%
悲观情景: 考虑医院采购预算紧缩 → CAGR = 5%(下降5个百分点)
影响: 拖累综合CAGR约1-2个百分点
```

**示例3: 集采降价**
```
基准情景: IVD业务CAGR = 15%,毛利率稳定
悲观情景: 集采大幅降价 → CAGR = 8%,毛利率下降5个百分点
影响: 拖累综合CAGR约2-3个百分点

| **12-15%** | **5.5-6.4** | **适度配置** ← 命中此区间 |
| 10-12% | 4.5-5.4 | 观望 |
| 8-10% | 3.5-4.4 | 观望 |
| 5-8% | 2.5-3.4 | 不推荐 |
| <5% | 0.5-2.4 | 不推荐 |

CAGR = 14.5% 命中区间：**12-15%**
对应评分区间：**5.5-6.4**

**Step 2: 线性插值计算**

```
评分 = 下限评分 + (CAGR - 下限CAGR) / (上限CAGR - 下限CAGR) × (上限评分 - 下限评分)

代入数值：
评分 = 5.5 + (14.5 - 12) / (15 - 12) × (6.4 - 5.5)
     = 5.5 + 2.5 / 3 × 0.9
     = 5.5 + 0.833 × 0.9
     = 5.5 + 0.75
     = 6.25
```

#### 8.1.3 验证检查

- ✅ CAGR = 14.5% ∈ [12%, 15%] 区间正确
- ✅ 评分 = 6.25 ∈ [5.5, 6.4] 区间正确
- ✅ 评级 = "适度配置" 与评分区间对应

#### 8.1.4 结论

**结论**：综合评分 = **6.3分**（四舍五入保留1位小数）

**投资评级**：☑️ 适度配置

**评级说明**：稳健成长型，适合追求中等收益、持有周期3-5年的投资者
```
