## 核心概念

传统情景分析（乐观/基准/悲观）只给出**3个离散情景**，但现实世界是连续的。蒙特卡洛模拟通过**大量随机模拟**，得到营收预测的**完整概率分布**。

**与传统情景分析的对比**:

| 维度 | 情景分析 | **蒙特卡洛模拟** |
|-----|---------|-----------------|
| **输出** | 3个确定值 | **完整概率分布** |
| **概率信息** | 离散概率（25%/50%/25%） | **连续概率密度** |
| **风险量化** | 粗略 | **精确（VaR、CVaR）** |
| **极端值** | 无法估计 | **可估计尾部风险** |
| **计算量** | 小 | **大（需1万+次模拟）** |
| **适用场景** | 一般情况 | **高风险、复杂模型** |

**为什么使用蒙特卡洛**:
1. **更完整的风险画像**: 不仅知道期望值，还知道置信区间
2. **发现非线性风险**: 变量间的相互作用可能产生意外结果
3. **支持复杂模型**: 多变量、非线性关系更容易建模
4. **更好的决策支持**: 基于概率分布而非点估计做决策

## 蒙特卡洛模拟框架

### 一、模型构建

#### 1.1 定义关键变量

**选择影响营收的关键变量**:

| 变量类别 | 变量名 | 分布类型 | 典型参数 | 适用行业 |
|---------|-------|---------|---------|---------|
| **市场需求** | 市场增长率 | 正态分布 | μ=8%, σ=3% | 所有行业 |
| **市占率** | 市场份额变化 | 均匀分布 | [-2%, +5%] | 所有行业 |
| **价格** | 产品价格变化 | 三角分布 | 低=-5%, 模=0%, 高=+3% | 产品驱动型 |
| **成本** | 原材料成本 | 对数正态 | μ=基准, σ=10% | 制造业 |
| **汇率** | 汇率波动 | 正态分布 | μ=当前, σ=5% | 出口企业 |
| **政策** | 政策影响因子 | 离散分布 | {利好:30%, 中性:50%, 利空:20%} | 政策敏感型 |

#### 1.2 变量分布设定

**正态分布（最常用）**:
```
X ~ N(μ, σ²)

参数设定:
- μ（均值）: 基准预测值
- σ（标准差）: 基于历史波动率或专家判断

示例:
市场增长率 ~ N(8%, 3%²)
表示:
- 68%概率落在 [5%, 11%]
- 95%概率落在 [2%, 14%]
- 99.7%概率落在 [-1%, 17%]
```

**均匀分布（范围明确但分布未知）**:
```
X ~ U(min, max)

适用场景:
- 市场份额变化
- 新产品渗透率
- 技术替代速度

示例:
市占率变化 ~ U(-2%, +5%)
表示在-2%到+5%之间等概率分布
```

**三角分布（有最可能值）**:
```
X ~ Triangular(low, mode, high)

参数:
- low: 悲观值
- mode: 最可能值
- high: 乐观值

适用场景:
- 价格变化
- 成本变化
- 新产品上市时间

示例:
价格变化 ~ Triangular(-5%, 0%, +3%)
表示最可能不变，下行风险大于上行
```

**对数正态分布（正值且右偏）**:
```
ln(X) ~ N(μ, σ²)

适用场景:
- 成本（不能为负，且可能大幅上升）
- 资产价格
- 销售额（已有基数上的增长）

示例:
原材料成本 ~ LogNormal(μ=当前成本, σ=10%)
```

**离散分布（类别变量）**:
```
政策情景:
- 利好: 概率30%，影响+10%
- 中性: 概率50%，影响0%
- 利空: 概率20%，影响-15%
```

#### 1.3 建立营收模型

**基础营收模型**:
```
营收 = 基期营收 × (1 + 市场增长率) × (1 + 市占率变化) × (1 + 价格变化)

或更复杂:
营收 = f(需求, 供给, 价格, 成本, 竞争, 政策, ...)
```

**示例：半导体设备企业模型**:
```python
def revenue_model(
    base_revenue,      # 基期营收
    market_growth,     # 市场增长率 ~ N(15%, 5%)
    market_share_gain, # 市占率提升 ~ U(0%, 5%)
    price_change,      # 价格变化 ~ Triangular(-5%, 0%, +3%)
    policy_factor,     # 政策因子 ~ {利好:30%, 中性:50%, 利空:20%}
    competition_factor # 竞争因子 ~ N(0%, 3%)
):
    # 计算新市占率
    new_share = base_share * (1 + market_share_gain)
    
    # 计算市场总需求
    market_size = base_market * (1 + market_growth)
    
    # 计算营收
    revenue = market_size * new_share * (1 + price_change)
    
    # 应用政策和竞争调整
    revenue = revenue * (1 + policy_factor) * (1 + competition_factor)
    
    return revenue
```

### 二、模拟执行

#### 2.1 单次模拟流程

```python
def single_simulation():
    # 1. 从分布中随机抽样
    market_growth = random.normal(8%, 3%)
    market_share = random.uniform(-2%, 5%)
    price_change = random.triangular(-5%, 0%, 3%)
    
    # 2. 代入模型计算营收
    revenue_2029 = revenue_model(
        base_revenue=100,
        market_growth=market_growth,
        market_share_gain=market_share,
        price_change=price_change
    )
    
    # 3. 计算CAGR
    cagr = (revenue_2029 / 100)^(1/5) - 1
    
    return {
        'revenue_2029': revenue_2029,
        'cagr': cagr,
        'market_growth': market_growth,
        'market_share': market_share,
        'price_change': price_change
    }
```

#### 2.2 大规模模拟

```python
def monte_carlo_simulation(n_simulations=10000):
    results = []
    
    for i in range(n_simulations):
        result = single_simulation()
        results.append(result)
    
    return results

# 执行1万次模拟
results = monte_carlo_simulation(10000)
```

**模拟次数选择**:
- 1,000次: 快速估算，误差较大
- 10,000次: 标准选择，平衡精度和速度
- 100,000次: 高精度，用于关键决策

#### 2.3 收敛性检验

**为什么需要收敛性检验**:
- 确保模拟次数足够
- 避免随机误差影响结论

**检验方法**:
```
方法1: 均值稳定性
- 将10,000次模拟分为10批，每批1,000次
- 计算每批的均值
- 如果10个批次的均值差异<1%，则认为收敛

方法2: 分位数稳定性
- 比较不同模拟次数下的P10/P50/P90
- 如果增加模拟次数后分位数变化<0.5%，则认为收敛
```

**风险指标**:
```
VaR (Value at Risk):
- 定义: 在X%置信水平下的最大损失
- VaR_90 = P10 = 10%概率营收低于此值

- 定义: 超过VaR阈值时的平均损失
- CVaR_90 = 营收<P10时的平均营收

下行风险:
- 概率(营收 < 基准情景)
- 概率(CAGR < 0)
```

## 实战案例

### 案例: 某半导体设备企业5年营收预测

#### 基础信息
- **当前营收**: 50亿元（2024年）
- **行业**: 半导体设备
- **市场**: 国产替代+全球扩产

#### 变量设定

| 变量 | 分布 | 参数 | 说明 |
|-----|------|------|------|
| 市场增长率 | 正态 | μ=20%, σ=8% | 行业高景气但波动大 |
| 市占率提升 | 均匀 | [0%, 8%] | 国产替代空间 |
| 价格变化 | 三角 | [-8%, 0%, +3%] | 竞争加剧，价格承压 |
| 地缘政治风险 | 离散 | {无影响:60%, 制裁:30%, 全面封锁:10%} | 关键风险 |

#### 营收模型

```python
def semiconductor_revenue_model(
    base_revenue=50,
    market_growth,      # ~ N(20%, 8%)
    share_gain,         # ~ U(0%, 8%)
    price_change,       # ~ Triangular(-8%, 0%, +3%)
    geo_risk            # ~ Discrete
):
    # 基础增长
    market_size = 1 + market_growth
    share = 1 + share_gain
    price = 1 + price_change
    
    # 地缘政治影响
    geo_impact = {
        '无影响': 1.0,
        '制裁': 0.85,      # 关键零部件受限
        '全面封锁': 0.50   # 业务萎缩
    }[geo_risk]
    
    # 计算营收
    revenue_2029 = base_revenue * market_size * share * price * geo_impact
    
    return revenue_2029
```

**龙卷风图结果**:

   - 关注地缘政治信号：提前减仓
   - 设置止损：如果营收不及P10（<58亿），重新评估

3. **置信区间预测**:
   - 保守预测（P25）: 78亿，CAGR 9%
   - 中性预测（P50）: 102亿，CAGR 15%
   - 乐观预测（P75）: 118亿，CAGR 19%

## 实施指南

### Python实现示例

```python
import numpy as np
from scipy import stats

def monte_carlo_revenue_forecast(
    base_revenue,
    n_simulations=10000,
    years=5
):
    """
    营收预测蒙特卡洛模拟
    """
    results = {
        'revenue_2029': [],
        'cagr': [],
        'market_growth': [],
        'share_gain': [],
        'price_change': []
    }
    
    for _ in range(n_simulations):
        # 从分布中抽样
        market_growth = np.random.normal(0.08, 0.03)  # N(8%, 3%)
        share_gain = np.random.uniform(-0.02, 0.05)   # U(-2%, 5%)
        price_change = np.random.triangular(-0.05, 0, 0.03)  # Tri(-5%, 0, 3%)
        
        # 计算营收
        revenue_2029 = base_revenue * (1 + market_growth)**years * \
                       (1 + share_gain) * (1 + price_change)
        
        # 计算CAGR
        cagr = (revenue_2029 / base_revenue) ** (1/years) - 1
        
        # 记录结果
        results['revenue_2029'].append(revenue_2029)
        results['cagr'].append(cagr)
        results['market_growth'].append(market_growth)
        results['share_gain'].append(share_gain)
        results['price_change'].append(price_change)
    
    return results

# 统计分析
def analyze_results(results):
    revenues = results['revenue_2029']
    
    analysis = {
        'mean': np.mean(revenues),
        'median': np.median(revenues),
        'std': np.std(revenues),
        'p10': np.percentile(revenues, 10),
        'p25': np.percentile(revenues, 25),
        'p50': np.percentile(revenues, 50),
        'p75': np.percentile(revenues, 75),
        'p90': np.percentile(revenues, 90),
        'prob_loss': np.mean([r < 100 for r in revenues])  # 低于基期的概率
    }
    
    return analysis

# 执行
results = monte_carlo_revenue_forecast(base_revenue=100)
analysis = analyze_results(results)

print(f"期望营收: {analysis['mean']:.1f}亿")
print(f"中位数: {analysis['median']:.1f}亿")
print(f"90%置信区间: [{analysis['p10']:.1f}, {analysis['p90']:.1f}]亿")
print(f"低于基期概率: {analysis['prob_loss']*100:.1f}%")
```

## 蒙特卡洛模拟过程展示规范 ⭐️ v2.6.0新增

### 核心原则

蒙特卡洛模拟章节必须采用**四段落结构**：
1. **假设依据** - 说明输入参数的概率分布设定
2. **计算过程** - 展示单次模拟示例和统计结果
3. **验证检查** - 验证模拟结果的合理性
4. **结论** - 给出P50、VaR等关键指标

### 示例模板

```markdown
### 7. 蒙特卡洛模拟

#### 7.1 假设依据

**输入参数概率分布**：

| 变量 | 分布类型 | 参数设置 | 设定依据 |
|------|---------|---------|---------|
| 市场增长率 | 正态分布 | μ=8%, σ=3% | 历史5年行业数据 |
| 市占率变化 | 均匀分布 | [-2%, +5%] | 竞争格局分析 |
| 价格变化 | 三角分布 | 低=-5%, 模=0%, 高=+3% | 定价策略评估 |
| 政策因子 | 离散分布 | 利好30%, 中性50%, 利空20% | 政策研判 |

**基期数据**：
- 基期营收：367亿
- 基期市场规模：2000亿
- 基期市占率：18.35%

#### 7.2 计算过程

**Step 1: 单次模拟示例（第1次）**

```
随机抽样结果：
- 市场增长率 = 9.2%（从N(8%,3%)抽样）
- 市占率变化 = +3.1%（从U(-2%,5%)抽样）
- 价格变化 = -1.5%（从三角分布抽样）
- 政策因子 = 0%（抽样结果：中性情景）

计算过程：
新市场规模 = 2000 × (1+9.2%) = 2184亿
新市占率 = 18.35% × (1+3.1%) = 18.92%
营收 = 2184 × 18.92% × (1-1.5%) = 404.2亿
```

**Step 2: 10,000次模拟统计结果**

| 统计指标 | 数值(亿) | 含义 |
|---------|---------|------|
| 均值 | 8,650 | 期望营收 |
| 标准差 | 1,820 | 波动程度 |
| P10 | 6,500 | 10%概率低于此值 |
| P25 | 7,500 | 25%概率低于此值 |
| **P50** | **8,800** | **中位数** |
| P75 | 10,200 | 75%概率低于此值 |
| P90 | 11,800 | 90%概率低于此值 |

**Step 3: 风险价值(VaR)计算**

```
VaR(95%) = 6,200亿

含义：有5%的概率，2029年营收将低于6,200亿
```

#### 7.3 验证检查

- ✅ P50 (8,800亿) 与情景分析基准情景 (8,000亿) 方向一致
- ✅ P10 (6,500亿) 与悲观情景 (5,000亿) 方向一致
- ✅ P90 (11,800亿) 与乐观情景 (10,000亿) 方向一致
- ✅ 均值 (8,650亿) 在P25-P75区间内

#### 7.4 结论

**结论**：
- P50（中位数）= **8,800亿**
- VaR(95%) = **6,200亿**
- 90%置信区间 = [6,500亿, 11,800亿]

**敏感性排序**：
1. 市场增长率：影响幅度 ±8.5%
2. 市占率变化：影响幅度 ±5.2%
3. 价格变化：影响幅度 ±3.1%
4. 政策因子：影响幅度 ±2.8%
```
