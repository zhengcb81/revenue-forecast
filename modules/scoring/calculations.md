######  4.2.4 综合营收与综合CAGR计算 ⭐️⭐️⭐️ v2.0.0新增核心章节

**核心概念**:通过加权平均三种情景的营收预测,得到"期望值",反映所有可能性的综合结果。

**计算公式**:

**1. 综合营收计算**:
```
2029年综合营收 = Σ(各情景2029年营收 × 情景概率)
              = 乐观营收×25% + 基准营收×50% + 悲观营收×25%
```

**2. 综合CAGR计算**:
```
综合CAGR = (2029年综合营收 / 2024年实际营收)^(1/5) - 1
```

**3. 年度综合营收(可选)**:
如果需要年度路径,可计算每年的综合营收:
```
第n年综合营收 = 乐观营收_n×25% + 基准营收_n×50% + 悲观营收_n×25%
```

**完整计算示例**:

**基础数据**:
- 2024年实际营收: 120亿元
- 乐观情景(25%):2029年营收200亿元,5年CAGR = (200/120)^(1/5)-1 = 10.8%
- 基准情景(50%):2029年营收160亿元,5年CAGR = (160/120)^(1/5)-1 = 5.9%
- 悲观情景(25%):2029年营收100亿元,5年CAGR = (100/120)^(1/5)-1 = -3.6%

**综合计算**:
```
2029年综合营收 = 200×25% + 160×50% + 100×25%
              = 50 + 80 + 25
              = 155亿元

综合CAGR = (155 / 120)^(1/5) - 1 = 5.2%
```

**综合CAGR的意义**:
- 反映了所有可能性的加权平均值
- 是投资决策的核心参考指标
- 已内含了风险(通过悲观情景)和收益(通过乐观情景)
- 避免了事后主观调整的双重计算

**情景概率的调整原则**:

**默认概率**: 乐观25% / 基准50% / 悲观25%

**调整因素**:

1. **确定性高** → 基准概率上调至60%
   - 订单可见性强(长单、国家战略订单)
   - 政策支持明确(医疗新基建、国产替代)
   - 竞争格局稳定(寡头垄断)

2. **护城河深** → 悲观概率下调至15-20%
   - 技术壁垒高(专利保护)
   - 品牌认知强(茅台定价权)
   - 供应链优势明显

3. **地缘政治风险高** → 乐观概率下调至15-20%
   - 美国制裁风险(半导体设备)
   - 核心零部件外购
   - 海外业务占比高

**示例**:
- **迈瑞医疗**(护城河深,确定性高): 乐观25% / 基准55% / 悲观20%
- **联影医疗**(地缘政治风险高): 乐观20% / 基准55% / 悲观25%
- **中微公司**(国家战略驱动,确定性极高): 乐观30% / 基准60% / 悲观10%

**注意事项**:
1. **概率总和必须等于100%**
2. **基准情景概率通常最高**(50-60%)
3. **极端情况才调整概率**(大多数情况使用默认25%/50%/25%)
4. **概率调整要有充分理由**(在报告中明确说明)

**输出格式**:

在报告中添加以下表格:

| 情景 | 2029年营收(亿) | 概率 | 加权营收(亿) | 5年CAGR |
|------|--------------|------|-------------|---------|
| 乐观 | XXX | 25% | XXX | XX% |
| 基准 | XXX | 50% | XXX | XX% |
| 悲观 | XXX | 25% | XXX | XX% |
| **综合** | **XXX** | **100%** | **XXX** | **XX%** |

**关键结论**:
- 综合CAGR = XX%(用于第八章评分)
- 综合营收 = XXX亿元(未来5年营收期望值)

## 综合CAGR计算过程展示规范 ⭐️ v2.6.0新增

### 核心原则

综合CAGR章节必须采用**四段落结构**：
1. **假设依据** - 说明情景概率设定的理由
2. **计算过程** - 展示加权平均的每一步计算
3. **验证检查** - 验证CAGR计算正确性
4. **结论** - 给出最终综合CAGR

### 示例模板

```markdown
### 4.4 综合CAGR计算

#### 4.4.1 假设依据

**情景概率设定**：

| 情景 | 概率 | 设定依据 |
|------|------|---------|
| 乐观 | 25% | 公司护城河深，但地缘政治风险存在 |
| 基准 | 50% | 最可能发生的情景 |
| 悲观 | 25% | 考虑外部不确定性 |

**各情景2029年营收**（来自4.1-4.3计算）：
- 乐观情景：870.8亿
- 基准情景：747.4亿
- 悲观情景：525.9亿

**基期营收**：2024年实际营收 = 367.0亿

#### 4.4.2 计算过程

**Step 1: 计算加权2029年营收**

| 情景 | 2029年营收(亿) | 概率 | 加权计算 | 加权营收(亿) |
|------|---------------|------|---------|-------------|
| 乐观 | 870.8 | 25% | 870.8×0.25 | 217.70 |
| 基准 | 747.4 | 50% | 747.4×0.50 | 373.70 |
| 悲观 | 525.9 | 25% | 525.9×0.25 | 131.48 |
| **合计** | - | 100% | 217.70+373.70+131.48 | **722.88** |

**Step 2: 计算综合CAGR**

```
综合CAGR = (2029年综合营收 / 2024年实际营收)^(1/5) - 1

代入数值：
综合CAGR = (722.88 / 367.0)^(1/5) - 1
        = (1.9697)^(0.2) - 1
        = 1.1452 - 1
        = 0.1452
        = 14.5%
```

#### 4.4.3 验证检查

**正向验证**：
```
367.0 × (1 + 14.5%)^5 = 367.0 × 1.9697 = 722.9亿 ✅
```

**反向验证**：
```
(722.88 / 367.0)^(1/5) - 1 = 14.52% ≈ 14.5% ✅
```

**合理性检查**：
- 综合CAGR (14.5%) 应在悲观 (7.5%) 和乐观 (18.9%) 之间 ✅
- 综合CAGR 应更接近基准情景 (15.3%)，因为基准概率最高 ✅

#### 4.4.4 结论

**结论**：综合5年CAGR = **14.5%**

**用途**：
- 此CAGR将用于第八章的综合评分计算
- 评分 = 查表(14.5%) = 6.3分
```
